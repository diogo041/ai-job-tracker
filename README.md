- Authentication
- Job application tracking
- Resume upload
- Analytics dashboards
- AI-powered resume/job matching
- VPS deployment
- CI/CD with GitHub Actions
- Production monitoring

- Node.js
- PostgreSQL
- Docker

- GitHub Actions
